# Olist-end-end-data-analytics
A Data Analytics comprehensive project analyzes Olist's marketplace
Olist is a Brazilian e-commerce marketplace that connects sellers and customers. This project analyzes Olist’s marketplace by integrating transactional e-commerce data with marketing funnel data used for seller acquisition.

The goal is to understand how marketing-driven seller onboarding impacts downstream business outcomes such as orders, revenue, delivery performance, and customer reviews.
